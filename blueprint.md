
# Project Overview

This is a Flutter application that will implement a login flow and a dashboard screen.

## Implemented Features

* Initial project setup.

## Current Plan

* Create a login screen with a "Sign in with Google" button.
* Show a "Bienvenido Usuario" popup on button click.
* Navigate to a dashboard screen after the popup.
* The dashboard screen will display:
    * User summary (points, league).
    * Modules in progress.
    * Recommended modules.
    * Profile and rankings access.
    * "Continuar práctica" and "Explorar módulos" buttons.
